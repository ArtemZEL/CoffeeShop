# Mini-Project: E-commerce for Tropical Coffee Beans

## About The Project

This mini-project is a web application (e-commerce store) for selling high-quality tropical coffee beans. The project's goal is to create a convenient and modern digital experience for coffee connoisseurs.

## Functionality & Coming Soon
This is a live project, and its functionality will be expanded. Active development is currently underway.

### Planned Features:
- **Product Catalog:** A complete list of coffee varieties with detailed descriptions.
- **Product Pages:** Dedicated pages for each coffee variety with photos, taste profiles, aroma notes, and price.
- **Shopping Cart System:** Ability to add products to the cart and edit the order.
- **Checkout Process:** A seamless checkout flow with delivery and payment method selection.

- **User authentication system**
